# MERN Stack Portfolio

A personal portfolio website featuring a 3D interactive hero section, animated projects gallery, and a fully functional contact form.

## Tech Stack

**Frontend:**
- React (Vite)
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js (@react-three/fiber)

**Backend:**
- Node.js
- Express
- MongoDB
- Nodemailer

## Getting Started

1. **Frontend**:
   ```bash
   cd Frontend
   npm install
   npm run dev
   ```

2. **Backend**:
   ```bash
   cd Backend
   npm install
   npm run dev
   ```
